# R66S 固件自动编译

基于 GitHub Actions 的 R66S 设备 OpenWrt 固件自动编译项目，支持 ImmortalWrt 和 LEDE 双分支。

## 内置插件

- AdGuard Home 广告拦截
- Docker 容器管理
- 系统监控 (Netdata)
- Web 终端 (ttyd)
- ZeroTier 虚拟局域网
- Passwall (支持多种协议)
- OpenClash
- SSR Plus+
- Mihomo
- Argon 主题 (默认)

## 默认设置

- **IP**: 192.168.100.1
- **用户名**: root
- **密码**: password
- **主题**: Argon
